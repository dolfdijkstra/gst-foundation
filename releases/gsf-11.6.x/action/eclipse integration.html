Eclipse/Grrovy/GSF integration

To use GSF with Groovy in eclipse there are some steps to take to make you more productive

  • Install eclipse Indigo (3.7) EE version
  • Install Groovy plugin from http://dist.springsource.org/release/GRECLIPSE/e3.7/

    Instructions on how to install the Groovy plugin can he found at http://groovy.codehaus.org/Eclipse+Plugin.

  • Install and launch Content Server Developers Tools (CSDT).

    Create a source folder from the WEB-INF/gsf-groovy.

    You now can edit all the groovy Action files from eclipse.
